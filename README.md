# OGTPred-ESM
**OGTPred-ESM** 是一个基于深度学习的蛋白质 **最适生长温度（Optimal Growth Temperature, OGT）预测工具**。  
本项目创新性地融合了 **ESM-2 蛋白质语言模型的深度序列表征** 与 **传统理化特征**，通过多模态特征融合策略构建了一个高精度的 OGT 预测模型。

---

## 核心特性

### 多模态特征融合
- **ESM-2 深度表征**：利用预训练的蛋白质语言模型提取富含进化信息的序列嵌入。
- **理化特征工程**：基于 [iFeature](https://github.com/Superzchen/iFeature) 工具包计算蛋白质的物理化学特性。
- **PCA 降维优化**：对高维理化特征进行智能降维，保留最具判别力的信息。

###  先进的模型架构
- 深度神经网络结构，结合 **批归一化（Batch Normalization）** 和 **Dropout 正则化**。
- **Optuna 自动化超参数优化**，实现最优模型搜索。
- **早停机制（Early Stopping）** 防止过拟合。
- 提供完整的训练可视化与性能评估指标。

##  代码结构与说明

| 文件 / 目录 | 功能说明 |
|--------------|-----------|
| `./embedding.py` | 使用 **ESM-2 模型** 对蛋白质序列进行嵌入表征，生成 `.pt` 格式的特征文件。 |
| `./ogt_ml_base.ipynb` | 执行基准机器学习训练与测试，比较传统算法性能。 |
| `OGTPred-ESM/script/02_pca_ifeature.py` | 最终模型训练脚本，融合 ESM 嵌入与理化特征，完成 PCA 降维与深度学习训练。 |

---

##  模型02_pca_ifeature.py输入文件要求

| 文件类型 | 格式 | 内容描述 |
|-----------|--------|----------|
| **ESM-2 嵌入文件** | `.pt` | ESM-2 模型生成的序列嵌入，形状为 `[序列数量, 嵌入维度]`。 |
| **理化特征文件** | `.csv` | 使用 iFeature 工具包计算的各种蛋白质理化性质。 |
| **标签文件** | `.csv` | 每条序列对应的 OGT 值，列名为 `'ogt'`。 |

##  模型特点概览

- **多模态输入融合**：结合 ESM 深度特征 + 理化特征。
- **自动化优化流程**：Optuna 调参 + Early Stopping。
- **降维与正则化**：PCA + BatchNorm + Dropout。
- **高鲁棒性**：多特征组合增强泛化性能。
